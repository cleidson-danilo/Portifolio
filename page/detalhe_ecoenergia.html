Descrição

EcoEnergia é uma calculadora web que permite ao usuário estimar o consumo de energia de aparelhos e ambientes, promovendo economia e consciência ambiental.

Objetivo: Tornar o cálculo de consumo acessível, visual e educativo.
Plataforma: Web responsiva, compatível com dispositivos móveis.
Status: Ativo e em evolução.
Diferenciais: Interface minimalista, modo escuro, gráficos dinâmicos e histórico local.

Recursos

  • Cálculo de consumo diário, mensal e anual
  • Alternância entre tema claro e escuro
  • Design responsivo e otimizado para mobile
  • Interface intuitiva e acessível
  • Gráficos de consumo
  • Salvar histórico local

Tecnologias

HTML5 CSS3 JavaScript Bootstrap LocalStorage

Desenvolvimento

O projeto foi desenvolvido com foco na experiência do usuário, utilizando HTML5, CSS3 e JavaScript para criar uma interface limpa e funcional. O Bootstrap foi empregado para garantir responsividade. O armazenamento local (LocalStorage) foi implementado para salvar o histórico do usuário sem necessidade de backend.
