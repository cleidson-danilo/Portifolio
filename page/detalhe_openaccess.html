OpenAccess

Plataforma de mapeamento interativo baseada em dados abertos para visualizar e classificar acessibilidade urbana em Pernambuco.

Descrição

O OpenAcess consolida dados da Overpass API para exibir pontos acessíveis, limitados e não acessíveis no estado de Pernambuco, permitindo consulta eficiente e inclusiva para usuários.

Prova de conceito com experiência responsiva para desktop e dispositivos móveis.

Funcionalidades

  • Busca por voz e texto
  • Cluster de marcadores com Leaflet
  • Classificação visual de acessibilidade
  • Favoritos com LocalStorage
  • Filtros por tipo e categoria

Tecnologias

HTML5 CSS3 JavaScript Leaflet Overpass API Vercel
